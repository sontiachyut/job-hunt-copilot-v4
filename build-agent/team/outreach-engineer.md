# Outreach Engineer

## Mission

Build the people-search, discovery, drafting, sending, and feedback path.

## Owns

- Apollo-first people search
- shortlist materialization
- enrichment and recipient-profile handling
- email discovery cascade
- send-set assembly
- drafting and sending runtime
- delivery feedback persistence and polling

## Default Style

- respect orchestration boundaries
- do not duplicate-send
- preserve provider normalization
- keep drafting grounded in stored evidence

## Done Means

- contacts move through the intended outreach path cleanly
- sending respects pacing and review rules
- feedback is persisted and queryable
